Marks

  • Each point on the graph represents one batted ball

Channels

  • Color
    • Represents the quantitative attribute of Scoring Value
    • This is a good mapping because the viewer can easily distinguish the sweet spot where exit velocity and launch angle combine to make home runs
  • Exit Velocity (X-axis)
    • Represents the quantitative attribute Exit Velocity which is how fast the ball comes off the bat
    • This is a good mapping because continuous so it works well on the x-axis to show relationship between the other axis
  • Launch Angle (Y-axis)
    • A diverging attribute corresponding to the angle the ball comes off the bat, above 0 is a ball flying up, and below 0 is a ball flying towards the ground
    • This is a good mapping because it's clearly shows how this attribute relates to the scoring value of a batted ball

Colormap

  • Orange: well hit ball, Purple: poorly hit ball, White: average hit
    • Represents the scoring value of a batted ball. More orange points indicates a well hit ball while more purple indicates balls that did not result in scores.
    • This is a good mapping because it very clearly shows where the sweet spot is to optimize scoring value as a function of batted ball exit velocity and launch angle.


















Marks

  • Each area (bar)

Channels

  • Horizontal Position
    • Bars horiztonal possition differentiates the movie that they represent
    • This is a good choice because different titles can easily be identified by moving along the x-axis
  • Vertical Position
    • Bars vertical possition represent the number of hours the movie was viewed within 28 days of release on Netflix
    • This is a good choice because it is easy to compare the height of bars and estimate the viewing hours
  • Color
    • The color of the bar indicates wether the movie is an English or non-English title
    • Red bars are English titles and blue bars are non-English titles

Colormap

  • The colormap for this visualization is categorical and differentiates between English and non-English titles
  • This is a good choice of color map as the categorical data can easily be differetiated since the hues are very different
